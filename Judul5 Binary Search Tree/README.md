Judul Program : Sistem Pengelolaan Data Mahasiswa

Program ini berfungsi sebagai sistem pengelolaan data mahasiswa yang dirancang untuk memudahkan pengguna dalam menyimpan, mencari, dan menampilkan data mahasiswa berdasarkan NIM. Program menyediakan fitur utama berupa penambahan data mahasiswa, pencarian data mahasiswa, penampilan data mahasiswa secara terurut, serta perhitungan jumlah mahasiswa yang tersimpan pada sistem. Dengan adanya fitur-fitur tersebut, proses pengelolaan data menjadi lebih terstruktur dan efisien.
